# KTANE 模块助手

English README: [README.md](README.md)

一个轻量级、纯浏览器运行的 **Keep Talking and Nobody Explodes** 模块辅助工具。

本项目完全基于前端（HTML/CSS/JavaScript），无需构建步骤，也没有第三方依赖。

## 功能

- 复杂线路（Complicated Wires）求解
- 线路序列（Wire Sequences）跟踪（12 根线，按出现次数计算）
- 记忆（Memory）模块辅助（5 个阶段，支持键盘快速录入）
- 密码（Passwords）候选过滤与下一步检查位推荐
- 摩斯电码（Morse Code）匹配、频率显示与字母参考表
- 中英文切换（`EN` / `中文`）
- 每个模块独立重置按钮

## 已支持模块

1. **复杂线路（Complicated Wires）**
- 设置每根线属性（红、蓝、星、LED）。
- 设置炸弹条件（序列号偶数、有并口、电池数 >= 2）。
- 每根线即时给出“剪 / 不剪”结果。

1. **线路序列（Wire Sequences）**
- 录入每根线的颜色和连接目标（A/B/C）。
- 应用会按颜色统计出现次数，并依据表格规则判断是否剪断。

1. **记忆（Memory）**
- 为每个阶段选择显示数字与按钮标签。
- 应用会给出操作，格式为 `P<位置> / <标签>`。
- 键盘快捷录入：输入 5 位数字（`显示 + 4 个标签`）后按 Enter。

1. **密码（Passwords）**
- 按位置输入可能字母。
- 应用显示符合条件的候选单词。
- 推荐下一步优先检查的位置，以缩小搜索范围。

1. **摩斯电码（Morse Code）**
- 输入字母子串筛选单词。
- 显示匹配词对应频率与摩斯编码。
- 可点击字母参考面板快速追加输入。

## 项目结构

- `index.html`：页面结构与模块模板
- `styles.css`：样式、布局与状态表现
- `script.js`：核心逻辑、查表数据、本地化与事件处理

## 快速开始

1. 克隆或下载本仓库。
1. 用浏览器打开 `index.html`。

无需安装任何依赖。

## 开发说明

这是一个静态项目，直接修改文件并刷新浏览器即可查看效果。

可选本地静态服务：

```bash
# Python 3
python -m http.server 8080

# Node (npx)
npx serve .
```

然后访问 `http://localhost:8080`（或终端中显示的地址）。

## 说明

- 本工具用于特定模块的加速查询与辅助判断，不替代游戏手册本体。
- 为了保持简单，数据与逻辑直接写在 `script.js` 中。
